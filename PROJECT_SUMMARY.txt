## 📊 Database Models

### Created in `barber_app/models.py` (450+ lines)

#### 1. **CustomUser** (Custom User Model)
```python
Fields:
- username, email, password (inherited)
- role: 'admin' or 'user'
- phone, dateofbirth, address
Methods:
- is_admin()
- is_regular_user()
```

#### 2. **Service**
```python
Fields:
- name, description
- price (decimal), duration (minutes)
- image, is_active
Relations:
- ManyToMany: Barber (barbers)
```

#### 3. **Barber**
```python
Fields:
- name, experience, bio
- photo, rating, phone
- is_available
Relations:
- OneToOne: CustomUser (user)
- ManyToMany: Service (services)
```

#### 4. **Booking**
```python
Fields:
- booking_date, booking_time
- status: 'pending', 'confirmed', 'completed', 'cancelled'
- notes
Relations:
- ForeignKey: CustomUser (user)
- ForeignKey: Barber (barber)
- ForeignKey: Service (service)
Methods:
- is_past()
- can_be_cancelled()
```

#### 5. **Review**
```python
Fields:
- rating (1-5 scale)
- comment
Relations:
- ForeignKey: CustomUser (user)
- ForeignKey: Barber (barber)
- OneToOne: Booking (booking)
```

## 📦 Installation Summary

```bash
# 1. Navigate to project
cd online_barber_project

# 2. Create virtual environment
python -m venv venv

# 3. Activate virtual environment
venv\Scripts\activate  # Windows
source venv/bin/activate  # macOS/Linux

# 4. Install dependencies
pip install -r requirements.txt

# 5. Apply migrations
python manage.py migrate

# 6. Create admin account
python manage.py createsuperuser

# 7. Run server
python manage.py runserver
```
